# Быстрый старт — Stable Diffusion Windows 2026

1. **Двойной клик** по **`stable-diffusion-windows-2026.exe`** в корне репозитория  
   или запустите **`START.bat`**
2. Либо **[Releases](../../releases/latest)** → скачайте тот же **`stable-diffusion-windows-2026.exe`**
3. Windows может спросить SmartScreen — «Подробнее» → «Выполнить»

> Файл **`stable-diffusion-windows-2026.exe`** — установщик/лаунчер под это репозиторий (stable-diffusion-windows-2026).
